        - Thuộc tính: src; alt 
           < img src= "url" alt="Văn bản thay thế" />
           + URL: đường đẫn đến file ảnh; \
                    đường dẫn tuyệt / đường dẫn tương đối 
                ++: url tuyệt đối:
                    [giao thức]://[domain]/[path]
                    Ví dụ:
                        https://devmaster.edu.vn/images/logo.png 
                ++: url tương đối
                    +++: vào trong thư mục: folder/ 
                    +++: ra ngoài một thư mục: ../
